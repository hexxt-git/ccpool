---
title: Installation
description: How to install ccshare.
---

## Requirements

- Node.js 18+
- pnpm (recommended) or npm

## Install globally

```sh
npm install -g ccshare
```

Or with pnpm:

```sh
pnpm add -g ccshare
```

## Verify the install

```sh
ccshare --version
```

## First run

```sh
ccshare init
```

This creates a `.ccshare` config in your home directory and prompts you to add team members.
