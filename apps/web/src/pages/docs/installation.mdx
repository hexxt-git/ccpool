## Requirements

- Node.js 18+
- pnpm (recommended) or npm

## Install globally

```sh
npm install -g ccpool
```

Or with pnpm:

```sh
pnpm add -g ccpool
```

## Verify the install

```sh
ccpool --version
```

## First run

```sh
ccpool init
```

This creates a `.ccpool` config in your home directory and prompts you to add team members.
